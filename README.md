# Mortal Kombat 3

## Description

A web app clone of the classic arcade game Mortal Kombat 3. The game is built using React and Typescript.

## Installation

1. Clone the repo
   ```sh
   git clone
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```sh
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Technologies Used

- React
- Typescript
- CSS Modules
